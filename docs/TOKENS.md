# AuthInk Design Tokens

## Colors

### Ink

```css
--authink-ink-900: #111111;
--authink-ink-800: #242424;
--authink-ink-700: #3D3D3D;
```

### Paper

```css
--authink-paper: #FAF9F6;
--authink-mist: #E8E6DF;
```

### Seal

```css
--authink-seal: #C0392B;
```

