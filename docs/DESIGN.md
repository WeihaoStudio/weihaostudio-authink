# WeihaoStudio AuthInk Design System

## Design Philosophy

AuthInk 是 WeihaoStudio 面向身份认证场景打造的东方美学设计系统。

核心理念：

> Ink remembers identity.
>
> 墨承身份，印证信任。

## Visual Direction

- 中国风
- 水墨画
- 极简克制
- 星辰意境
- 企业级安全产品质感

## Brand Language

视觉元素：

- 水墨月相
- 山水留白
- 星辰
- 篆刻印章

整体避免传统国潮堆砌，保持现代 SaaS 产品的简洁与高级感。

## Product Scope

覆盖：

- Keycloak 登录主题
- OAuth 登录
- MFA / TOTP 验证
- 密码重置
- 错误状态

## Design Principles

1. 内容优先
2. 留白优先
3. 品牌一致
4. 安全可信
5. 可工程化落地
