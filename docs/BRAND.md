# AuthInk Brand Guideline

> WeihaoStudio 已确认的水墨月山星辰 Logo 为 AuthInk 品牌核心资产。

相关文档：[`DESIGN.md`](./DESIGN.md) · [`COLORS.md`](./COLORS.md) · [`TYPOGRAPHY.md`](./TYPOGRAPHY.md)

## Logo

### Logo 构成

| 元素 | 描述 |
|---|---|
| 月相墨迹 | 水墨笔触的月弧轮廓，品牌核心图形 |
| 山水意境 | 远山剪影，营造空间感与留白 |
| 星辰元素 | 星点 / 微光，连接数字世界意象 |
| 水墨字标 | WeihaoStudio 手写水墨风格文字 |

### Logo 版本

| 版本 | 用途 |
|---|---|
| Full Logo | 月相 + 星辰 + WeihaoStudio 字标 |
| Icon | 纯图形标记，无字标 |
| Light | 浅色版本，用于暗色背景 |
| Dark | 深色版本，用于浅色背景 |
| Mono | 单色版本，用于打印 / 单色输出 |

## Brand Anchor

印章元素承担：

- **身份确认** - 认证动作的视觉隐喻
- **品牌锚点** - 提供红色视觉记忆点
- **东方书画落款感** - 连接品牌与产品气质

印章红色来自 `--authink-color-seal`（浅色 `#c0392b` / 暗色 `#d45548`），是整个水墨体系中唯一的暖色强调。

> 详细色彩体系见 [`COLORS.md`](./COLORS.md)。

## Logo Usage

| 场景 | 推荐版本 |
|---|---|
| 登录页 | Full Logo |
| 导航栏 | Icon + Wordmark |
| favicon | Icon |
| 暗色背景 | Light Logo |
| 打印 / 单色 | Mono Logo |

### 间距

Logo 周围保留不小于标志高度 1/4 的安全空间，不可被其他元素侵入。

### 尺寸

- 最小使用尺寸：Icon 24px，Full Logo 高度 32px
- 登录页推荐：Full Logo 高度 40-48px

## Visual Language

### Ink Flow

关键词：自然、流动、克制、留白

水墨感来自：

- 淡墨层次的柔和渐变
- 笔触边缘的自然过渡
- 大面积留白形成的呼吸感
- 低饱和度色彩体系

### 品牌字标

WeihaoStudio 字标是品牌字形资产，**禁止用字体替代**。必须使用 Logo SVG / PNG 资产。

`--authink-font-brand`（`"LXGW WenKai", "Noto Serif SC", serif`）可用于文字还原的回退方案，但生产环境优先使用 Logo 资产。

> 详细字体系统见 [`TYPOGRAPHY.md`](./TYPOGRAPHY.md)。

## Do Not

禁止：

- 厚重毛笔字风格
- 过度国潮装饰（龙纹、金色、繁复纹样）
- 复杂山水插画
- 非品牌字体替代 Logo 字标
- 高饱和度色彩
- 改变 Logo 比例、旋转、添加描边

## Brand Tone

| 维度 | 定位 |
|---|---|
| 气质 | 文气、克制、高级 |
| 情绪 | 平静、可信、不轻浮 |
| 语言 | 简洁、直接、有分寸 |
| 参考品牌 | Linear、Vercel、Stripe、Apple |
