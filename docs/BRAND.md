# AuthInk Brand Guideline

## Logo

使用 WeihaoStudio 已确认水墨月山星辰 Logo。

包含：

- 月相墨迹
- 山水意境
- 星辰元素
- WeihaoStudio 水墨字标

## Brand Anchor

印章元素承担：

- 身份确认
- 品牌锚点
- 东方书画落款感

## Usage

| 场景 | Logo |
|---|---|
| 登录页 | Full Logo |
| 导航 | Icon + Wordmark |
| favicon | Icon |
| 暗色背景 | Light Logo |
| 单色输出 | Mono Logo |

## Do Not

禁止：

- 厚重国潮风
- 复杂装饰
- 非品牌字体替代 Logo
